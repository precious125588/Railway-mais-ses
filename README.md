# Prezzy — WhatsApp Session Pairing by PRECIOUS x

25 songs, crypto fix, deploy on Render.

## Deploy
1. Push to GitHub
2. Render → New Web Service → connect repo
3. Build: `npm install && npm run build`
4. Start: `node server.js`
5. Env: `NODE_VERSION=20`
6. Disk: `/opt/render/project/src/prezzy_auth` 1GB
